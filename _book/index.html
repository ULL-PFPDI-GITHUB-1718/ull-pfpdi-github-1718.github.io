Instrucciones para los autores de este libro

  • El libro está alojado en el repo https://github.com/ULL-PFPDI-GITHUB-1718/ull-pfpdi-github-1718.github.io
  • Instale git
  • Clone el libro en su máquina: git clone https://github.com/ULL-PFPDI-GITHUB-1718/ull-pfpdi-github-1718.github.io.git
  • Instale NodeJS
  • Instale gitbook en su ordenador
  • Instale gulp globalmente: npm i -g gulp
  • Instale las dependencias de este libro/proyecto npm i
  • Hágase miembro de GitHub
  • Escriba en markdown los textos que desee
  • Compile el libro con gulp build para producir el HTML
  • Para desplegar su version en GitHub escriba gulp deploy
  • Visite https://ull-pfpdi-github-1617.github.io/ para ver el resultado
  • Para mas detalles mira la sección Markdown y GitBook
